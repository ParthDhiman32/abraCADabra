# AbraCADabra

# Zine

# Description

This is our Gesture Control Device which uses PAJ7620 to detect gestures and it has upto 15 gestures, We have added 2 modes in this which is music mode and Work mode which help us to do shortcuts like changing tabs, muting volume, switching songs easily with just a move from your hand.

# How do we use it?

As we know that the world is coming to a end and your keyboard would be destroyed too and imagine if you need to use your keyboard or soup wants to do it then you can use our AbraCADabra which has 2 modes and 15 gestures for each mode and we can use it either while listening music or when doing some productive work.

# Why we made this?

We made this so that when you wanna change music or switch tabs you dont have to do it manually you can just glide your hand above the sensor and then the work is done and the thing is it is really easy to change the shortcuts you want manually in the python code.

1) First download the Code Folder from the Repo.
2) Download Pycharm and make a new project.
3) Now go to the Settings after making a new project, You can refer to the screenshot below for location.
